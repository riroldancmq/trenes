⚙️ Configuración Inicial

Para sincronizar datos entre dispositivos, necesitas un token de GitHub.

¿No tienes un token?
1. Ve a GitHub → Settings → Tokens
2. Click en "Generate new token (classic)"
3. Marca "repo" y genera el token
4. Copia el token y pégalo arriba

Nota: El token se guarda solo en tu navegador.
